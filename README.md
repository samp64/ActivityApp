# Hoop Activity App


## Deploying

```
git clone https://github.com/samp64/App.git
```

### Backend

Install Serverless

```
npm install -g serverless
```

Navigate to /api

Install dependencies

```
npm install 
```

### Frontend

Install dependencies

In root of the project:
```
npm install
```
